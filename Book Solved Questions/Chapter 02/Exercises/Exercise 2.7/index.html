Exercise 2.7

Write a short XHTML document that demonstrates the six different heading elements and that also includes normal paragraph text. Compare the fonts used for the various headings with the default font used for paragraph text in Mozilla 1.4.

Answer

The following XHTML markup demonstrates the six different heading elements. It also compares the fonts used for the headings with the default font used for paragraphs in Mozilla 1.4:

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
    <title>Web Technologies Exercise 2.7</title>
    <style>
        body { font-family: Arial, sans-serif; background-color: #f4f4f4; }
        h1, h2, h3, h4, h5, h6 { color: #333; }
        p { color: #666; }
    </style>
</head>
<body>
    <h1>This is a Heading 1</h1>
    <h2>This is a Heading 2</h2>
    <h3>This is a Heading 3</h3>
    <h4>This is a Heading 4</h4>
    <h5>This is a Heading 5</h5>
    <h6>This is a Heading 6</h6>
    <p>This is a normal paragraph with the default font in Mozilla 1.4, which is typically a serif font like Times New Roman.</p>
</body>
</html>
