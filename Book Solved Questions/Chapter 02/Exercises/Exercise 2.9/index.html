Exercise 2.9

If two displays have the same physical size but one has higher resolution than the other, then the same image displayed on both displays with the same width and height in pixels will appear how: smaller on the higher resolution display, larger on the higher resolution display, or the same size on both displays? Explain.

Answer

If two displays have the same physical size but one has a higher resolution, the same image displayed on both displays with the same width and height in pixels will appear smaller on the higher resolution display.

Here's why:

  • Resolution refers to the number of pixels in a given physical area. A higher resolution means more pixels are packed into the same physical space.

  • If the displays have the same physical size, but one has a higher resolution, the higher resolution display will have smaller individual pixels.

  • Therefore, the image, which takes up the same number of pixels, will occupy less physical space on the higher resolution display because the pixels are smaller, making the image appear smaller.

  • On the lower resolution display, the pixels are larger, so the image will take up more physical space, making it appear larger.
