Exercise 2.4

Write XHTML markup that will display three paragraphs (each “paragraph” can contain just a few words of text). Write the markup so that there is more vertical space between the second and third paragraphs than there is between the first and second.

Answer

The following XHTML markup meets the given requirements:

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
    <title>Exercise 2.4</title>
    <style>
        .extra-space { margin-top: 40px; }
    </style>
</head>
<body>
    <p>This is the first paragraph.</p>
    <p>This is the second paragraph.</p>
    <p class="extra-space">This is the third paragraph with extra space above.</p>
</body>
</html>

Result

This is the first paragraph.

This is the second paragraph.

This is the third paragraph with extra space above.
