Exercise 2.10

Using only XHTML as covered in this chapter, explain how you could provide a web page that contains an image that can be distorted by page visitors. Specifically, by manipulating their browsers, users should be able to stretch the image horizontally. The image may appear distorted when it is initially displayed.

Answer

To create a web page where visitors can distort an image horizontally by manipulating their browsers, you can utilize the <img> element in XHTML along with CSS. The key idea is to set a fixed width for the image and allow users to change the width using the browser's developer tools or resize the browser window. Here's how it can be done:

  • The image element is displayed with a fixed width and an automatic height, so it will stretch horizontally while keeping the aspect ratio intact initially.

  • However, if the user manipulates the width of the image by using browser tools or resizing the window, they can stretch the image horizontally, causing distortion.

  • By setting the width property in CSS (using percentage or pixels), users can increase or decrease the image's width, which will make the image appear distorted, especially when it is stretched beyond its natural aspect ratio.

Below is an example of XHTML markup with an image that users can stretch horizontally:

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
    <title>Stretchable Image Example</title>
    <style>
        img {
            width: 100%;
            height: auto;
        }
    </style>
</head>
<body>
    <h2>Stretchable Image</h2>
    <img src="image.jpg" alt="Image to stretch" />
</body>
</html>

When displayed, the image will initially appear with its full width, and users can stretch it horizontally by adjusting the browser width or through browser tools, causing the image to distort horizontally.
