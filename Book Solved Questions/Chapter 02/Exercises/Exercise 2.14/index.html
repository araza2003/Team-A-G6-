Question:

Rewrite the following example taken from the HTML 4.01 specification so that it com- plies with the requirements of the XHTML 1.0 standard as discussed in this chapter. The table produced by your version should look exactly the same as the table produced by the original version in a browser compliant with HTML 4.01.

<TABLE border="1"
        summary="This table gives some statistics about fruit
            flies: average height and weight, and percentage
            with red eyes (for both males and females).">
            
<CAPTION><EM>A test table with merged cells</EM></CAPTION>
<TR><TH rowspan="2"><TH colspan="2">Average
    <TH rowspan="2">Red<BR>eyes
<TR><TH>height<TH>weight
<TR><TH>Males<TD>1.9<TD>0.003<TD>40%
<TR><TH>Females<TD>1.7<TD>0.002<TD>43%
</TABLE>

Answer

The provided HTML code has been updated to comply with XHTML 1.0 standards by ensuring proper closing of tags, proper use of lowercase for attributes, and self-closing of tags like <br />.

Here is the corrected code shown above.

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
    <title>Fruit Flies Statistics</title>
</head>
<body>
    <table border="1" summary="This table gives some statistics about fruit flies: average height and weight, and percentage with red eyes (for both males and females).">
    <caption><em>A test table with merged cells</em></caption>
    <tr>
        <th rowspan="2"></th>
        <th colspan="2">Average</th>
        <th rowspan="2">Red<br />eyes</th>
    </tr>
    <tr>
        <th>height</th>
        <th>weight</th>
    </tr>
    <tr>
        <th>Males</th>
        <td>1.9</td>
        <td>0.003</td>
        <td>40%</td>
    </tr>
    <tr>
        <th>Females</th>
        <td>1.7</td>
        <td>0.002</td>
        <td>43%</td>
    </tr>
    </table>
</body>
</html>
