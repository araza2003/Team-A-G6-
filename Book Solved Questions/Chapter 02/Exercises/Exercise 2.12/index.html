Exercise 2.12

You are writing an HTML document that will reside in the App1 subdirectory of the Tomcat webapps/ROOT directory. The App1 directory contains a directory named legal. Write a relative URL that could be used within a document contained in App1 to refer to a document named copyright.html in the legal directory. Write a second relative URL that could be used within copyright.html to refer to an image file named logo.jpg in the App1 directory.

Answer

The following two relative URLs can be used as described in the exercise:

  • Relative URL from a document in App1 to copyright.html in the legal directory: legal/copyright.html

  • Relative URL from copyright.html (in the legal directory) to logo.jpg in the App1 directory: ../logo.jpg

Explanation:

  • For the first relative URL, since the document is in the App1 directory and the legal directory is a subdirectory of App1, the relative URL to copyright.html is simply legal/copyright.html.

  • For the second relative URL, from copyright.html (which is in the legal directory), to the logo.jpg image in the App1 directory, we use ../logo.jpg because ../ means to go up one directory level, which takes us back to App1, where the image is located.
