rel Attribute

Description
The `rel` attribute specifies the relationship between the current document and the linked document. It is commonly used for:
  • noopener: Prevents the new page from having access to the previous page's window object (used for security when opening links in new tabs).
  • noreferrer: Prevents the browser from sending the referring URL to the linked page.
  • stylesheet: Specifies that the linked document is a CSS stylesheet.
Example
<a href="https://example.com" rel="noopener">Visit Example</a>
